docs: Adiciona plano de implementação para módulo de licitações

Cria o documento de planejamento detalhado para o novo módulo avançado de licitações.

O plano abrange a arquitetura da solução, o modelo de dados completo, os fluxos de usuário para as funcionalidades de busca, análise com IA, checklists de tarefas, notificações e análise de compatibilidade.

A implementação é dividida em três fases para um desenvolvimento iterativo e gerenciável.
