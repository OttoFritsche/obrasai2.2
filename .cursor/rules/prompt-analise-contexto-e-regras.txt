**Tarefa:** Analisar Arquivos de Contexto e Regras do Projeto

**Objetivo:**
Preciso que você leia, analise e compreenda profundamente o conteúdo dos seguintes arquivos para internalizar o contexto do projeto ObrasAi e as regras/diretrizes que devemos seguir:

1.  `contexto_obrasai.md`: Descreve a visão geral, arquitetura, módulos, detalhes específicos, funcionalidades implementadas e próximos passos do projeto. **Este arquivo define O QUÊ estamos construindo e ONDE estamos.**

4.  regras-projeto.mdc : apresenta regras de boas praticas feito no lovable.dev onde começamos o projeto
**Instrução Principal:**
Sua tarefa é processar integralmente esses três arquivos. A partir deste momento, todo o seu trabalho e respostas relacionadas a este projeto devem estar estritamente alinhados com as informações, regras e contexto definidos neles. Considere-os como sua principal fonte de verdade para o projeto ObrasAI.
